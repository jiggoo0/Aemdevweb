# 🛠️ PRODUCT DEFINITION: AEMDEVWEB AI-INFRASTRUCTURE SYSTEM

## 1. Vision & Goals
โปรเจกต์นี้มุ่งเน้นการสร้าง **High-Performance Web Infrastructure** ที่ขับเคลื่อนด้วย **Next.js 16 (App Router)** และ **React 19** โดยมีหัวใจหลักคือระบบ **Autonomous AI Agent** ที่ฝังตัวอยู่ใน `root/.gemini/` เพื่อทำหน้าที่เป็น "สมองส่วนกลาง" ในการพัฒนาและดูแลรักษาโปรเจกต์แบบอัตโนมัติ

### Core Objectives:
- **Maximum AI Autonomy:** ระบบ AI สามารถตัดสินใจและดำเนินการพัฒนาได้เองเกือบ 100% ตามคำสั่งเดียว (Directives)
- **State-Aware Continuity:** AI จดจำบริบทและสถานะงานที่ค้างไว้ได้เสมอผ่าน `PROJECT_STATE.md` (Continuity Protocol)
- **Autonomous SEO Engine:** จัดการ Schema.org, Metadata และ E-E-A-T โดยอัตโนมัติ 100% เพื่อผลลัพธ์ทางเทคนิคที่แม่นยำที่สุด

## 2. Key Features & AI Capabilities
- **Self-Evolving AI Skills:** AI มีความสามารถในการติดตั้งและอัปเดต Skills ให้สอดคล้องกับฟีเจอร์ใหม่ๆ ใน Next.js 16, Tailwind CSS v4 และเครื่องมือสมัยใหม่
- **Precision Validation (AIPC):** ทุกการเปลี่ยนแปลงต้องผ่านการรัน `pnpm aipc` เพื่อยืนยันความสมบูรณ์ 100% (Format, Lint, Type-Check, Build)
- **Proactive Auditing:** AI ตรวจสอบประสิทธิภาพ (LCP, Web Vitals) และ Caching Strategy (Next Cache Components) อย่างต่อเนื่อง

## 3. Targeted Performance & ROI
- **LCP Optimization:** เป้าหมายความเร็วในการโหลดเนื้อหาหลัก < 1 วินาที (Zero-FOUC Architecture)
- **Conversion Focus:** ระบบ SEO & Performance ต้องสนับสนุนการปิดการขายได้จริง (B2B / SME Specialized)
